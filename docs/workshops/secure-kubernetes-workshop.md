# Secure Kubernetes Storage Workshop

**Date:** 20 July 2026  
**Time:** 11:00–16:00  
**Location:** University of Cambridge - WestHub (room D)  
**Format:** In person with online participation

## Overview

This workshop explores approaches for protecting sensitive data stored in
Kubernetes block volumes.

The workshop will cover automated LUKS encryption, Kubernetes Operators,
CSI-based encryption, HashiCorp Vault, federated identity and key lifecycle
management.

## Intended audience

The workshop is suitable for:

- Kubernetes administrators;
- cloud and platform engineers;
- Trusted Research Environment operators;
- research infrastructure engineers;
- information security professionals;
- developers working with sensitive data.

## Learning objectives

Participants will learn how to:

1. identify storage risks in Kubernetes research environments;
2. encrypt Cinder-backed block volumes using LUKS;
3. compare Operator-based and CSI-based encryption;
4. store encryption keys securely in HashiCorp Vault;
5. implement tenant-specific Vault policies;
6. use OIDC claims for federated access control;
7. plan key rotation and deletion workflows.
